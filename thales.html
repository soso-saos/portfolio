Projet Thales Alenia Space

Développement d’un site web dynamique pour un banc avionique intelligent en collaboration avec Thales Alenia Space.

Résumé du projet

Ce projet a été réalisé dans le cadre d’un partenariat avec Thales Alenia Space. L’objectif était de concevoir un site web interactif en PHP permettant de piloter et superviser un banc avionique intelligent.
Le travail a inclus la gestion de la sécurité, l’intégration d’outils de visualisation et la rédaction d’une documentation technique complète.

Compétences mobilisées

  • Développement web (PHP, HTML, CSS, JavaScript)
  • Gestion de projet en équipe
  • Cybersécurité et gestion des accès
  • Rédaction de documentation technique
