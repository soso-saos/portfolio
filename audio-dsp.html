Projet Ingénierie : Station de Mixage & DSP Audio

Interface graphique interactive pour le pilotage temps réel de flux audio industriels.

1Le Défi

Développer une application capable de manipuler des signaux audio en temps réel (changement de vitesse, spatialisation, gain) sans latence perceptible. L'enjeu était d'interfacer un langage de haut niveau (Python) avec un moteur multimédia bas niveau performant.

2La Stack Technique

  • Langage : Python 3.
  • Moteur Audio : GStreamer 1.0 (Framework pipeline multimédia).
  • Interface (GUI) : ipywidgets (Intégration Web/Jupyter).
  • Système : Module subprocess pour le pilotage des processus système et bibliothèque glob pour la gestion dynamique des fichiers.

3Résultats

L'application finale permet le transcodage et le ré-échantillonnage de formats audio à la volée. J'ai conçu une architecture modulaire qui permet d'ajouter de nouveaux effets DSP (Digital Signal Processing) sans refondre le cœur de l'application.
